# Business Portfolio Website

A modern, full-stack business portfolio built to showcase services, projects, and professional presence online. This application combines a responsive frontend with a scalable backend and a cloud-hosted database.


**Frontend**

* HTML5, CSS3
* Tailwind CSS
* React (with TypeScript)

**Backend**

* Node.js
* Express (TypeScript)

**Database**

* Supabase (PostgreSQL)

---

## ✨ Features

* Responsive and modern UI using Tailwind CSS
* Dynamic content powered by React + TypeScript
* RESTful API with Node.js and Express
* Secure and scalable database with Supabase
* Project showcase section
* Contact form / business inquiry system
* Easy to customize and extend

---

## 📁 Project Structure

```
/client        → React frontend
/server        → Node.js + Express backend
/database      → Supabase configuration (if applicable)
```

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Setup frontend

```
cd client
npm install
npm run dev
```

### 3. Setup backend

```
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create `.env` files in both `client` and `server` directories.

Example variables:

```
SUPABASE_URL=your_supabase_url
SUPABASE_ANON_KEY=your_supabase_key
PORT=5000
```
